# Depósito Stock - Control de Inventario

Esta es una mini aplicación para el control de stock de un depósito. Permite gestionar productos, registrar entradas y salidas, y realizar el seguimiento de envíos a sucursales.

## Características

- **Gestión de Stock:** Visualización en tiempo real del stock disponible.
- **Movimientos:** Registro detallado de entradas (compras/producción) y salidas (ventas/envíos).
- **Control de Sucursales:** Seguimiento específico de productos enviados a cualquiera de las 3 sucursales.
- **Reportes Excel:** Generación de reportes en formato XLSX para stock actual, historial de movimientos y envíos por sucursal.
- **Categorización:** Clasificación de productos por categorías (Sillones, Mesas, Sillas, Macetas, Decoración, etc.).
- **Diseño Moderno:** Interfaz responsiva y amigable optimizada para dispositivos móviles.

## Tecnologías Utilizadas

- HTML5 / CSS3 (Variables CSS, Flexbox, Grid)
- JavaScript Vanilla (ES6+)
- LocalStorage para persistencia de datos local.
- [SheetJS (XLSX)](https://sheetjs.com/) para la generación de reportes Excel.

## Uso

La aplicación funciona completamente en el navegador y guarda los datos de forma local en el dispositivo. No requiere de una base de datos externa para su funcionamiento básico.
